trait WebstartOptions
extends scala.NotNull
Defines the configurable parameters for the webstart task.

Method Summary
abstract def jnlpFile : Path
The location to write the .jnlp file to. It must be in 'webstartOutputDirectory'.
abstract def jnlpXML (jars : scala.Seq[WebstartJarResource]) : scala.xml.Elem
Generates the .jnlp file using the provided resource descriptions. Each resource description provides the path of the jar relative to 'webstartOutputDirectory' and whether or not it is the main jar.
abstract def webstartExtraLibraries : PathFinder
Libraries external to the project needed for webstart. This is mainly for scala libraries.
abstract def webstartGzip : Boolean
If true, gzip compression will be applied to all jars. If pack200 compression is enabled, gzip compression is also applied to the archives with pack200 compression. A version of each file without gzip compression is still created in the webstart output directory.
abstract def webstartLibDirectory : Path
The location to put all jars that are not the main jar. It must be in 'webstartOutputDirectory'.
abstract def webstartLibraries : PathFinder
The libraries needed for webstart. Note that only jars are used; directories are discarded.
abstract def webstartMainJar : Path
The main jar to use for webstart.
abstract def webstartOutputDirectory : Path
The location to put all generated files for webstart.
abstract def webstartPack200 : Boolean
If true, pack200 compression is applied to all jars (main and libraries). A version of each jar without pack200 compression is still created in the webstart output directory.
abstract def webstartResources : PathFinder
Resources to copy to the webstart output directory.
abstract def webstartSignConfiguration : scala.Option[SignConfiguration]
If defined, configures signing of jars. All jars (main and libraries) are signed using this configuration.
abstract def webstartZip : scala.Option[Path]
If defined, this specifies where to create a zip of the webstart output directory. It cannot be in the output directory.
