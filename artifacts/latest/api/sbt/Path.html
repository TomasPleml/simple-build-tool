override def ## : Path
Creates a base directory for this path. This is used by copy and zip functions to determine the relative path that should be used in the destination. For example, if the following path is specified to be copied to directory 'd', ((a / b) ##) / x / y the copied path would be d / x / y The relativePath method is used to return the relative path to the base directory.
