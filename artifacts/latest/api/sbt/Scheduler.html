trait Scheduler[D]
extends scala.NotNull with AnyRef
Schedules work of type D. A Scheduler determines what work is ready to be processed. A Scheduler is itself immutable. It creates a mutable object for each scheduler run.

Starts a new run. The returned object is a new Run, representing a single scheduler run. All state for the run is encapsulated in this object.
