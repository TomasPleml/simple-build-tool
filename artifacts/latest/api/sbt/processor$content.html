Simple Build Tool 0.7.3 API

This document is the API specification for Simple Build Tool 0.7.3 API

Class Summary
abstract class BasicProcessor extends Processor with AnyRef
An interface for code that operates on an sbt `Project` but doesn't need to modify command processing.
sealed trait Command extends scala.NotNull
Data type representing a runnable command related to processor management.
class CommandParser (errorMessage : java.lang.String, defParser : DefinitionParsing) extends CommandParsing with AnyRef
Parses commands. `errorMessage` is the String used when a command is invalid. There is no detailed error reporting. Input Strings are assumed to be trimmed.
trait CommandParsing extends scala.NotNull
Parses a command String
class CommandRunner (parser : CommandParsing, execute : Executing) extends AnyRef
Parses and executes a command (connects a parser to a runner).
final class DefineProcessor (val pdef : ProcessorDefinition) extends Command with AnyRef
A command to add the given processor definition.
final class DefineRepository (val repo : RepositoryDefinition) extends Command with AnyRef
A command to register the given repository to be used for obtaining `Processor`s.
sealed trait Definition extends scala.NotNull
class DefinitionParser extends DefinitionParsing with AnyRef
Parses the String representation of definitions.
trait DefinitionParsing extends scala.NotNull
Parses a definition `String`.
class Execute (manager : Manager, info : Info, log : Logger) extends Executing with AnyRef
Executes a parsed command.
trait Executing extends scala.NotNull
Executes a parsed command.
final class Exit (val code : Int) extends ProcessorResult with AnyRef
class Handler (baseProject : Project) extends scala.NotNull with AnyRef
trait Info extends scala.NotNull
Prints information about processors.
class InfoImpl (manager : Manager, prefix : java.lang.String, parser : CommandParser, print : (java.lang.String) => Unit) extends Info with AnyRef
class Loader extends scala.NotNull with AnyRef
trait Manager extends scala.NotNull
Manages the processor and repository definitions.
class ManagerFiles (val retrieveBaseDirectory : java.io.File, val retrieveLockFile : java.io.File, val definitionsFile : java.io.File) extends AnyRef
Files needed by ManagerImpl. `retrieveBaseDirectory` is the directory that processors are retrieved under. `retrieveLockFile` is used to synchronize access to that directory. `definitionsFile` is the file to save repository and processor definitions to. It is usually per-user instead of per-project.
class ManagerImpl (files : ManagerFiles, scalaVersion : java.lang.String, persist : Persist, log : Logger) extends Manager with AnyRef
class ParsedProcessor (val label : java.lang.String, val processor : Processor, val arguments : java.lang.String) extends scala.NotNull with AnyRef
class Persist (val lock : GlobalLock, lockFile : java.io.File, defParser : DefinitionParser) extends Persisting with AnyRef
trait Persisting extends scala.NotNull
Handles serializing `Definition`s.
trait Processor extends scala.NotNull
An interface for code that operates on an sbt `Project`.
final class ProcessorDefinition (val label : java.lang.String, val group : java.lang.String, val module : java.lang.String, val rev : java.lang.String) extends Definition with AnyRef
final class ProcessorException (val message : java.lang.String, cause : java.lang.Throwable) extends java.lang.RuntimeException with AnyRef
An exception used when a `Processor` wants to terminate with an error message, but the stack trace is not important. If a `cause` is provided, its stack trace is assumed to be important.
sealed trait ProcessorResult extends scala.NotNull
The result of a Processor run.
final class Reload (insertArgs : java.lang.String*) extends ProcessorResult with AnyRef
final class RemoveDefinition (val label : java.lang.String) extends Command with AnyRef
A command to remove the processor or repository definition currently associated with the given `label`. If the definition is associated with other labels, those are not affected.
final class RepositoryDefinition (val label : java.lang.String, val url : java.lang.String) extends Definition with AnyRef
class Retrieve (retrieveDirectory : java.io.File, module : ModuleID, lock : GlobalLock, lockFile : java.io.File, repositories : scala.Seq[Resolver], log : IvyLogger) extends scala.NotNull with AnyRef
final class Success (val project : Project, val onFailure : scala.Option[java.lang.String], insertArgs : java.lang.String*) extends ProcessorResult with AnyRef
