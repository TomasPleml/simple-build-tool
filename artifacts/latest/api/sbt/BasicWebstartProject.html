[source: sbt/BasicWebstartProject.scala]
abstract class BasicWebstartProject
extends BasicScalaProject with WebstartScalaProject with WebstartOptions with WebstartPaths
Defines default implementations of all methods in WebstartOptions except for jnlpXML. packageAction is overridden to create a webstart distribution after the normal package operation.
