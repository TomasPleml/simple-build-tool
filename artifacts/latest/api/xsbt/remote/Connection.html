abstract def execute (command : java.lang.String) : Boolean
Tell the remote client to execute the given command. Waits for the client to return "true" or "false" indicating whether the command succeeded or failure. This result is then passed on to the caller of this method. Exceptions from the underlying I/O code are not handled.
