abstract class BasicProcessor
extends Processor with AnyRef
An interface for code that operates on an sbt `Project` but doesn't need to modify command processing.
Method Summary
override final def apply (label : java.lang.String, project : Project, onFailure : scala.Option[java.lang.String], args : java.lang.String) : ProcessorResult
Apply this processor's action to the given `project`. The arguments are passed unparsed as a single String `args`. The return value optionally provides additional commands to run, such as 'reload'. Note: `project` is not necessarily the root project. To get the root project, use `project.rootProject`. The `label` used to call the processor is provided to allow recursing.
