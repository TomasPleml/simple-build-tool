class CommandParser(errorMessage : java.lang.String, defParser : DefinitionParsing)
extends CommandParsing with AnyRef
Parses commands. `errorMessage` is the String used when a command is invalid. There is no detailed error reporting. Input Strings are assumed to be trimmed.
